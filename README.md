# REAL OR FAKE

딥페이크 위험성과 개인정보 보호의 중요성을 주제로 제작한  
캠페인 웹사이트입니다.

---

## 📌 프로젝트 개요
- 주제: 딥페이크 예방 캠페인
- 목표: SNS에 공개된 얼굴 이미지와 영상이 딥페이크 제작에 악용될 수 있다는 점을 알리고, 개인정보 보호의 필요성을 전달
- 구성: 메인 / 딥페이크 악용 사례 / 예방 수칙 / 캠페인 영상

---

## 💻 주요 작업

### 🧾 웹사이트 제작
- HTML / CSS / JavaScript 기반의 원페이지 캠페인 사이트 제작
- 고정형 헤더와 섹션 이동 메뉴 구현
- PC와 모바일 환경에 맞춘 반응형 레이아웃 적용

---

### 🎬 캠페인 영상 연결
- JavaScript를 활용해 영상 모달 기능 구현
- 메인 버튼과 헤더 버튼을 통해 캠페인 영상 확인 가능
- 모달 닫기 시 영상이 멈추도록 설정

---

### 🔐 딥페이크 예방 콘텐츠 구성
- 얼굴 이미지와 영상이 딥페이크에 악용될 수 있는 사례 정리
- SNS 이용자가 쉽게 이해할 수 있도록 위험 흐름 구성
- 공개 범위 설정, 영상 업로드 주의 등 실천 가능한 예방 수칙 안내
